# Каталог ID-сервисов России (2025)

Отечественный рынок аутентификации разделился на пять независимых контуров. Интеграция с ними решает разные задачи: от простого входа на сайт до юридически значимого документооборота.

## 1. Государственные и инфраструктурные
| Провайдер | Назначение | Доступность для внешних сайтов | Особенности интеграции |
| :--- | :--- | :--- | :--- |
| **ЕСИА (Госуслуги)** | Юридически значимые действия, получение госуслуг | Закрытый контур | Требует подключения через ИС, обоснования сбора ПДн, часто использование ГОСТ-криптографии. |
| **ЕБС (Биометрия)** | Подтверждение личности по лицу и голосу | Жесткий комплаенс | Только для банков, нотариата и высокорисковых операций. |
| **ЕМИАС** | Медицинские данные | Изолированный | Доступ только для авторизованных медучреждений Москвы. |
| **Мир ID (НСПК)** | Платежные сервисы, программы лояльности | B2B партнерства | Завязан на экосистему карт Мир. |

## 2. Технологические гиганты и Телеком (Низкий порог входа)
| Провайдер | Назначение | Доступность для внешних сайтов | Особенности интеграции |
| :--- | :--- | :--- | :--- |
| **Яндекс ID** | Быстрый вход (SSO), доступ к данным Яндекса | Открытый (OAuth 2.0) | Самый простой старт. Достаточно зарегистрировать приложение в консоли разработчика. |
| **VK ID** | Авторизация через соцсеть | Открытый (OAuth 2.0) | Быстрое подключение. Дает доступ к социальному графу пользователя. |
| **Мобильный ID** | Беспарольный вход по номеру телефона | Коммерческий API | Работает на уровне операторов (МТС, Мегафон, Билайн). Пользователь жмет кнопку на экране телефона. |
| **МТС ID** | Вход в экосистему МТС | API по запросу | Выгоден при пересечении аудиторий с сервисами оператора. |
| **Ключ Ростелеком**| Доступ к умному дому и домофонам | Закрытый/B2B | Интеграция для застройщиков и управляющих компаний. |

## 3. Банковские системы (Высокий уровень доверия)
| Провайдер | Назначение | Доступность для внешних сайтов | Особенности интеграции |
| :--- | :--- | :--- | :--- |
| **T-ID (Т-Банк)** | Идентификация с передачей проверенных данных | Открытый (для ИП/ООО) | Отличная документация. Автоматизированный процесс подключения. Выдает подтвержденный номер телефона. |
| **Сбер ID** | Доступ к крупнейшей базе клиентов РФ | Корпоративный API | Долгая интеграция. Жесткие требования к безопасности сайта партнера. |
| **Альфа ID** | Вход для клиентов банка | B2B партнерства | Ориентирован на e-commerce и финтех-сервисы. |
| **ВТБ ID** | Вход для клиентов ВТБ | B2B партнерства | Аналог Сбера, но с меньшим охватом рынка. |
| **Газпром ID** | Вход в сервисы Газпром-Медиа (Rutube, Premier) | Внутренний/B2B | Фокус на медийные активы и партнеров холдинга. |

## 4. Отраслевые (Ритейл и Логистика)
| Провайдер | Назначение | Доступность для внешних сайтов | Особенности интеграции |
| :--- | :--- | :--- | :--- |
| **Почта.ID** | Получение посылок без паспорта | Внутренний API | Связан с единой системой почтовых отделений. |
| **СДЭК ID** | Выдача отправлений по SMS | Внутренний API | Упрощает логистику внутри одной компании. |

| **X5 ID** | Карты лояльности Пятерочки и Перекрестка | B2B партнерства | Данные о покупках и профиль покупателя офлайн-ритейла. |
| **РЖД ID** | Покупка билетов, программа лояльности | Внутренний API | Изолированная система транспортной монополии. |

## 5. Корпоративные решения (B2B IdM/IAM)
Эти системы не используются для логина на публичных сайтах. Они управляют правами сотрудников внутри корпоративной сети. Интегрировать их в обычный интернет-магазин нельзя.

* **Solar inRights (Ростелеком-Солар):** Тяжелый Enterprise IdM. Заменяет западные Oracle IAM и SailPoint.
* **Avanpost (IDM, FAM, Web SSO):** Полный стек управления доступом. Умеет работать с ГОСТ-алгоритмами.
* **1IDM:** Легковесное решение для автоматизации выдачи прав сотрудникам.
* **Индиго:** Управление привилегированными пользователями (PAM). Замена CyberArk.
